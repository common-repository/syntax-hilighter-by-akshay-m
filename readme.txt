Best Syntax Highlighting Plugin available for Wordpress that allows to display and Highlight Code Snippets in various Languages on your blog.

== Description ==

The need of displaying Code Snippets in your blog is fulfilled by the Plugin effectively.

The plugin can be used as follows, 
Code that you want to have syntax highlighted and displayed in Code Textarea, should be encoded with pre and code tags as discussed in Installation section of the Plugin.

This plugin can be used to display and highlight code of various languages including C, CPP, Java, PHP, Ruby, Markup (HTML) etc.

== Installation ==
From your WordPress dashboard

1. Visit 'Plugins > Add New'
2. Search for 'Syntax Highlighter by Akshay M.'
3. Activate 'Syntax Highlighter by Akshay M.' from your Plugins page. 

How to Use:

* Write your code in tags like,
   <pre>
   &lt;pre class=&quot;language-ruby&quot;&gt;&lt;code class=&quot;language-ruby&quot;&gt;
   // Sample Code of Ruby
   &lt;/code&gt;&lt;/pre&gt;</pre>
* Including Line numbers for your code in the codearea,
   <pre>
   pre class=&quot;language-ruby line-numbers&quot;&gt;&lt;code class=&quot;language-ruby&quot;&gt;
   // Sample Code of Ruby
   &lt;/code&gt;&lt;/pre&gt;</pre>
* Example for using with various languages:

Ruby -
<pre>&lt;pre class=&quot;language-ruby&quot;&gt;&lt;code class=&quot;language-ruby&quot;&gt; # Ruby Code &lt;/code&gt;&lt;/pre&gt;</pre>
Java - 
<pre>&lt;pre class=&quot;language-java&quot;&gt;&lt;code class=&quot;language-java&quot;&gt; # Java Code &lt;/code&gt;&lt;/pre&gt;</pre>
PHP - 
<pre>&lt;pre class=&quot;language-php&quot;&gt;&lt;code class=&quot;language-php&quot;&gt; # PHP Code &lt;/code&gt;&lt;/pre&gt;</pre>
C -
<pre>&lt;pre class=&quot;language-c&quot;&gt;&lt;code class=&quot;language-c&quot;&gt; # C Code &lt;/code&gt;&lt;/pre&gt;</pre>
CPP -
<pre>&lt;pre class=&quot;language-cpp&quot;&gt;&lt;code class=&quot;language-cpp&quot;&gt; # CPP Code &lt;/code&gt;&lt;/pre&gt;</pre>
HTML -
<pre>&lt;pre class=&quot;language-html&quot;&gt;&lt;code class=&quot;language-html&quot;&gt; # HTML Code &lt;/code&gt;&lt;/pre&gt;</pre>
Markup -
<pre>&lt;pre class=&quot;language-markup&quot;&gt;&lt;code class=&quot;language-markup&quot;&gt; # Markup Code &lt;/code&gt;&lt;/pre&gt;</pre>

== Frequently Asked Questions ==

= How to use for Java language? =
   Use following syntax for implementing thr plugin for Java Language.
   <pre>
   &lt;pre class=&quot;language-java&quot;&gt;&lt;code class=&quot;language-java&quot;&gt;
      // Sample Code of Ruby
   &lt;/code&gt;&lt;/pre&gt;
      # Similarly use for other languages</pre>
   For using it for other languages, replace 'language-ruby' with 'language-java'
